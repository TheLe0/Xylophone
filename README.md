# Xylophone

![Android CI](https://github.com/TheLe0/Xylophone/workflows/Android%20CI/badge.svg)

Simulator of a xylophone, for play some songs.

![Print](./img/print.png)

It's very simple, you only click on a button and is going to play a musical note.

* Red: C
* Orange: D
* Yellow: E
* Turquoise: F
* Green: H
* Blue: A
* Purple: B
